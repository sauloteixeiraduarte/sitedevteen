  • Dúvidas?

  • Conseguir maior destaque dentre os estudantes para completar as vagas no projeto, além de conseguir informações acerca do acesso a tecnologia dos alunos, além de conhecimentos adjacentes a área da informática para um futura comparação ao fim do projeto.

  • Incentivar os estudantes a participar mais assiduamente do projeto Dev Teen e instigá-los a pesquisar mais sobre o ramo profissional da informática.

  • Introduzir o básico de lógica computação aos estudantes, para que saibam mais sobre sistemas operacionais, consigam atualizar e baixar drives e outros arquivos de instalação.

  • Desenvolver conhecimento técnico sobre tipos de linguagem que serão estudadas para codificar e executar linhas de código utilizando ferramentas como o FreeCodeCamp e Visual Studio.

  • Expor, praticar e entender sobre a estrutura básica de um código HTML, como o uso de tags head, body e outros elementos padrões.

  • Aprimorar a digitação de conteúdo html com o uso de estilos diferentes de títulos, subtítulos e parágrafos; além de promover o uso de listas numeradas e não-numeradas para organização de tópicos.

  • Fazer com que os alunos que frequentam o projeto possam manipular aspectos da imagem utilizando diversos atributos de HTML, como máscaras, densidade, dimensões, etc.

  • Fazer com que os alunos desenvolvam a capacidade de interligar seções internas e páginas externas.

  • Estimular, nos estudantes, a habilidade de criar menus e botões para ligar páginas criadas pelos mesmos para facilitar o uso e a clareza na construção de seus websites.

  • Desenvolver conhecimentos teóricos e práticos sobre a linguagem de modelagem CSS, para quê ela é usada e como é ligado um documento CSS a um código HTML. Além disso, os estudantes participantes do projeto terão irão adquirir experiência de divisão de seções utilizando containers e tags especiais.

  • Estimular o uso contínuo do CSS como uma linguagem facilitadora de modificação no estilo (cores, bordas, etc).

  • Proporcionar a aprendizagem de como realizar diferentes tipos de alinhamento de textos e elementos HTML usando o CSS; além de desenvolver atividades sobre disposição de conteúdo em containers.

  • Proporcionar momentos de interação em equipes, dentro e fora do laboratório de informática para que possam codificar utilizando as propriedades HTML e CSS.

  • Realizar a avaliação dos sites produzidos pelos participantes do projeto.

  • Avaliar nível de inglês, raciocínio lógico e matemática dos estudantes e adquirir informação sobre qualificação do projeto.

  • Utilizar os dados coletados pelos questionários aplicados ao corpo discente, e a partir destes, desenvolver conclusões acerca dos resultados do projeto aplicado na Escola Estadual Almeida Cavalcanti.
